## 下载

### Docker

```
docker pull apachecn0/pyqt4-doc-zh
docker run -tid -p <port>:80 apachecn0/pyqt4-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g pyqt4-doc-zh
pyqt4-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
